# DigiMarket — Цифровой маркетплейс в Telegram

## Структура проекта

```
marketplace/
├── index.html          # Главная страница
├── catalog.html        # Каталог товаров
├── sellers.html        # Продавцы
├── how.html            # Как это работает
├── admin.html          # Админ-панель (веб)
├── style.css           # Стили
├── app.js              # Скрипты главного сайта
├── admin.js            # Скрипты админ-панели
├── bot/
│   ├── bot.py          # Telegram бот
│   ├── requirements.txt
│   └── .env.example    # Пример .env файла
└── api/
    ├── server.js       # API сервер
    ├── package.json
    └── data/           # Данные (создаётся автоматически)
```

## Запуск

### 1. Веб-сайт

Просто откройте `index.html` в браузере или запустите локальный сервер:

```bash
# Python
cd marketplace
python -m http.server 3000

# Node.js
npx serve .
```

### 2. API сервер

```bash
cd api
npm install
npm start
```

API будет доступен на `http://localhost:3001`

### 3. Telegram бот

```bash
cd bot
pip install -r requirements.txt
cp .env.example .env
# Заполните .env файл
python bot.py
```

### 4. Настройка бота

1. Получите токен бота у [@BotFather](https://t.me/BotFather)
2. Узнайте свой Telegram ID у [@userinfobot](https://t.me/userinfobot)
3. Заполните `.env` файл:

```
BOT_TOKEN=ваш_токен
ADMIN_IDS=ваш_id
WEB_APP_URL=http://localhost:3000
API_URL=http://localhost:3001
```

## Админ-панель

### Веб-админка

Откройте `admin.html` в браузере. Функции:

- 📊 Статистика (пользователи, товары, продажи)
- 📢 Массовая рассылка (текст, фото, видео)
- 👥 Управление пользователями
- 🛍 Управление товарами
- ⚙️ Настройки (комиссия, приветствие, тех.работы)

### Бот-админка

Отправьте `/admin` боту. Функции:

- 📊 Статистика
- 📢 Рассылка
- 🛍 Товары
- 👥 Пользователи
- ⚙️ Настройки
- 🌐 Ссылка на веб-админку

## Команды бота

- `/start` — Запуск бота
- `/admin` — Админ-панель (только для админов)
- `/help` — Помощь
- `/cancel` — Отмена текущего действия
